# Kotlin Todo List App

Простое консольное приложение для управления списком задач.

## Функциональность
- Добавление задач
- Просмотр всех задач
- Отметка задач как выполненных
- Удаление задач
- Сохранение в файл

## Технологии
- Kotlin
- File I/O
- Collections

## Запуск
```bash
kotlinc src/Main.kt -include-runtime -d TodoApp.jar
java -jar TodoApp.jar
```
